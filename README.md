# Fast-Campus-Git-Practice
Practice repo of fast-campus, 14th Jan to 15th Jan
![GitHub commit activity](https://img.shields.io/github/commit-activity/w/lonerkim/fast-campus-git-practice)
## Pre-Requititest
All practices are tested in Debian 13 (trixie) with 6.17.4-2-pve kernel.

- Python^3.9
- GCC^14.2.0
- Git
- vim

## How to Start
```bash
$ git clone https://github.com/lonerkim/fast-campus-git-practice
$ cd fast-campus-git-practice
$ python3 main.py
```
## Installation
```bash
apt update && apt install -y git
# install python if you need
# install your choice of editor in my case I used VIM which is not included in Proxmox LXC turnkey package.
# apt install -y vim
```
## Features
This will greet user multiple times using `print()` and `for` with even number detection.
